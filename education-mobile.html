Education History

I graduated from Shailer Park State High School in 2023. Since then, I've become an undergraduate at Griffith Universitys' Bachelor of Computer Science. Detailed here are notable achievements, classes, and assessments that will be most relevant to my career.


Some classes may not include details of assessments in accordance with Griffith University security policy.

  • 1811ICT - Programming Principles

    Programming Principles involved the fundamentals of Python programming. Included is file manipulation, basic Object Oriented Programming, more advanced problem solving and debugging techniques, and proper practice for researching issues whilst programming.


  • 1011ICT - Applied Computing

    Applied Computing taught me about SQL and Database Management Systems. Skills here included advanced database structuring using normalisation and denormalisation techniques, use of SQL to create and host databases online, and team management.


  • 1007ICT - Computer Systems and Cybersecurity

    This class focused on the fundamentals of how computers and basic networking functions under the hood, at the binary level and at more abstract levels.


    The major assignment for this course was the design of a circuitboard for stocktaking - stock levels are increased and decreased based on the months selected (represented by a 4 bit value - 0000 for January, 0001 for February, and so on) and then stored in a memory circuit.

    Pictured are the circuitboard diagrams. Skills taught in this class include abstract electrical engineering, networking, and boolean logic, all of which are extremely useful when programming with low-level languages such as C due to the advanced control over data locations.


  • 1808ICT - Discrete Structures

    This class was a deep dive on Discrete Mathematics - set and graph theory (and their relevant algorithms) up to mathematical proofs. These topics are directly designed to cover the most vital mathematics and logic for programming.


  • 1004ICT - Professional Practice in IT

    Professional Practice in IT was a course designed to improve my soft skills in the IT world - workplace interactions, team dynamics, and leadership. Assessments included prediction of the future evolution of the field and a career plan.

  • 1017SCG Foundation Mathematics and 1013ICT Mathematics for Computer Science

    Together, these classes taught me Calculus, Probability, Statistics, and Algebra. Furthermore, they developed my use of the C programming language to solve problems relating to these topics.

    Attached are my submissions for each of 4 questions. Together, these programs performed; sums & derivatives of polynomials, basic statistical analysis, and various applications relating to matrices including multiplication.


    These classes developed all of the fundamental mathematical skills for a wide variety of applications, honed my critical thinking skills, and introduced me to C and low-level programming languages like it.
