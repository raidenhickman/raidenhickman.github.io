  • Markov Chain Text Generation

    One of my most recent projects, for which the GitHub files are

    here

    , is a program which breaks down any input text into statistics for how often certain words come after other sets of words. I used this technology to attempt to guess the word that follows a phrase the user types in. Also included is a nonsense text generator that attempts to match the patterns of the training data. However, be warned; only 4 books were used as training data, so the results won't be very accurate.


    This project taught me more about the fundamentals of AI and autocomplete programs, maximising the efficiency of a program in a short time period, and maintaining the flexibility of a program; the same base code was used as a base for both included sample programs.
