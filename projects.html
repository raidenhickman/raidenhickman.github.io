  • Weather Tracker API


    The project, linked here, is a website which queries a weather API - openweathermap.org - for a location you can enter. The weather data for the area is then displayed automatically.



    Attached above is a screenshot of the website in action, after entering Brisbane, QLD, Australia as the location and Celsius as the measurement.


    This project taught me about how to query APIs, use virtual environments, create and utilise .env files, and how to handle security protocols with sensitive information like API keys in public or open source projects while also ensuring they are as useable as possible to the end user.

  • Markov Chain Text Generation

    One of my most recent projects, for which the GitHub files are here, is a program which breaks down any input text into statistics for how often certain words come after other sets of words. This is used for nonsense text generation and, more interestingly, attempting to complete sentences entered by the user.



    Included above is a sample output for the sentence completion algorithm, with four classic books as input (listed in the GitHub readme file). Higher accuracy comes with more training data, of course.


    This project taught me more about the fundamentals of AI and autocomplete programs, maximising the efficiency of a program in a short time period, and maintaining the flexibility of a program; the same base code was used as a base for both included sample programs.
