  • Weather Tracker API


    The project, linked here, is a website which queries a weather API - openweathermap.org - for a location you can enter. The weather data for the area is then displayed automatically - including temperature, wind speed, and an icon based on the sky at the time.



    Attached above is a screenshot of the website in action, after entering Brisbane, QLD, Australia as the location and Celsius as the measurement.


    This project taught me about how to query APIs, use virtual environments, create and utilise .env files, and how to handle security protocols with sensitive information like API keys in public or open source projects while also ensuring they are as useable as possible to the end user.

  • Markov Chain Text Generation

    One of my most recent projects, for which the GitHub files are here, is a Markov chain algorithm, which breaks down input text to a list of which words come after which sets of other words most often. This has a variety of applications, as demonstrated in the sample program file.



    Included above is a sample output for a sentence completion algorithm, with four classic books as input (listed in the GitHub readme file).


    This project taught me more about the fundamentals of AI and autocomplete algorithms and maintaining flexibility while optimising; the same base code was used for both included sample programs, and could be used for most Markov chain related applications with little alteration..
